Pokébase Android Application

May 2016

Gotta catch 'em all! Pokébase for Android is an application that allows you to see information about all 721 Pokémon! Using Google Cloud Platform running App Engine with Cloud SQL, I developed this application for an introductory database class along with fellow classmate Brittany Berlanga. See the new update as of July 2016!

Make Your Own Account

Users can create their own account and will be automatically be added to the database.

Search the Database for Any of the 721 Pokémon

Using Endpoints with Google Cloud Platform, the application calls custom API methods to query Pokémon by Region and/or Type.

See Pokémon Profiles in an Instant

Data such as type, region, height, weight, base experience, and move sets can be queried from the database as well.

Create Teams to Keep Track of Your Pokémon

Users can add customized teams to the database to keep track of their own Pokémon.

Customize Pokémon Stats

Give your Pokémon a nickname, level, and customize what moves they know.

Easily View All of Your Created Teams

Users are given an easy to use Recycler View to see all of their teams and add new ones.

Pokébase Update

July 2016

Completely localized the application, making the transition over to a SQLite database. Rewrote the back-end database connector and also implemented some major UI overhauls including a new Pokémon profile page as well as new animations. Pokébase is now available on GitHub!

Completely Revamped Pokémon Profile Page

Now with new animations, easily view more information about all 721 Pokémon! Play Pokémon cries and easily switch between the previous and next Pokémon as well.

Redesigned Pokémon Team View

Newly designed Pokémon team view with new animations and small UI tweaks.
