Who Done Did It?close Oct – Dec 2015

This product will enable users to play the "Who Done Did It?" card game through a computer interface that allows offline play as well as server-connected play with up to 5 players. With the continued rise of multiplayer online games, traditional card games have yet to make a shift from playing in-person to playing through a computer interface. This product will allow fans of this traditional card game to play against others online, breaking the barriers of in-person play. As the lead UI designer, implemented a Swing GUI in Java for an online multiplayer game based on the original card game Clue and had a hand in implementation of backend game logic modules in a 6 membered team.
