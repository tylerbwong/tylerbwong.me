EPIC SLO Android Application

Apr 2016 - Present

Win. Build. Send. This mobile application for Android allows users to login with Facebook, see upcoming and past events, and even submit and view prayer requests. Currently working on this as a way to teach myself Android App Development.

Works with Facebook

Users can log into the application with Facebook, allowing an easy transfer of information.

Drawer Layout with Navigation View

With the implementation of a Drawer Layout using a Navigation View, users can easily switch between pages.

Prayer Feed

This dynamic prayer feed uses a Recycler View to allow the user to scroll through incoming prayer requests. Prayer requests are pulled from the cloud (Firebase).

Submit Prayer Requests

Users can submit prayer requests as themselves or anonymously, to the cloud, where other users can view and keep track of. Past personal submissions can also be viewed in the adjacent tab.

Events

Upcoming and past events can easily be viewed with a simple Recycler View using a Card Layout.
